## 📌 프로젝트 소개  
학부생 시절 ‘게임 프로그래밍’ 과목의 기말 프로젝트(1인)입니다.  
DirectX9을 활용하여 그래픽을 구현하고 C++를 사용하여 개발하였습니다.

## 📌 조작법  
- 키보드  
    - 좌/우 키 → 새 캐릭터 이동  
    - 스페이스바 → 나의 현재 HP와 적 HP 확인  
    - ESC → 게임 종료  
- 마우스  
    - 왼쪽 클릭 → 화면 이동  
    - 오른쪽 클릭 → 화면 고정
